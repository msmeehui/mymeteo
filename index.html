    Data Sources
    Forecast data:
    Open-Meteo Forecast API
    Location autocomplete:
    Open-Meteo Geocoding API
    Current-location names:
    OpenStreetMap Nominatim
    Rain radar animation:
    Buienradar
    Near-term rain chance:
    Open-Meteo adjusted with Buienradar point rain data in the Netherlands for the first 2 hours, then radar data toward the 8-hour limit. Rain intensity is shown separately with a small three-step meter.
    Radar frame decoding:
    gifuct-js through esm.sh
    Fallback radar tiles:
    LibreWXR
    Map tiles:
    OpenStreetMap through Leaflet
    Analytics:
    Simple Analytics for privacy-friendly usage statistics.
    Make Shortcut

    Add MyMeteo to your phone's home screen so it opens like an app.

    iPhone or iPad

    1. Open MyMeteo in Safari.
    2. Tap the Share button.
    3. Tap Add to Home Screen.
    4. Keep the name "MyMeteo" or choose your own name, then tap Add.

    Android

    1. Open MyMeteo in Chrome.
    2. Tap the More menu.
    3. Tap Install app or Add to Home screen.
    4. Confirm by tapping Install or Add.

    After that, you can open MyMeteo from the icon on your home screen.
